Production Tips

📐 Static Posters

Use your Content Studio (HTML tool) for Jobs of the Week and Career Tips. Use ChatGPT image gen for custom posters (Coming Soon, Launch Day, Feature Showcase).

🎬 Motion Graphics in CapCut

Import static screenshots → add zoom/pan (ken burns) → add text with "slam" or "typewriter" animations → add trending audio → export 1080×1920 vertical.

⏰ Batch Production

Create all static posters in 1 session. Create all 7 motion graphics in 1-2 CapCut sessions. Film all 3 on-camera videos in one sitting (change shirts between takes).

📱 Best Posting Times

Tanzania: 6-9 PM EAT weekdays, 10AM-12PM weekends. TikTok: daily. IG feed: 3-4x/week. Stories: 2-3/day. Schedule IG posts via Meta Business Suite (free).

🎵 CapCut Audio Tips

Use CapCut's built-in audio library. Search "reveal", "motivation", "bass drop" for impact sounds. Match bass hits to text slams. Keep audio trending — check TikTok for current sounds.

📊 Content Mix Rationale

6 static + 7 motion + 3 video = 50/50 split. Motion graphics get higher engagement than static on TikTok. Static carousels get highest saves on Instagram. Video builds trust.
